# Applied Data Analytics

This repository contains a curated portfolio of applied data analytics projects focused on
extracting actionable insights from real-world, operational datasets.

The work emphasizes:
- Robust data cleaning and validation
- Statistically responsible analysis and feature selection
- Clear, decision-oriented visualizations
- Reproducible Python workflows

Projects are organized by domain as the portfolio grows.  
Current work focuses on analytics conducted within academic institutional contexts.

---

## Domains

### Academia
Applied analytics projects centered on curriculum effectiveness, student outcomes, program pathways,
and institutional decision support using real LMS and administrative data.

Additional domains (healthcare, finance) will be added as future projects are completed.

---

## Tools & Techniques

- **Languages:** Python  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn  
- **Methods:** Correlation analysis (Spearman & Pearson), feature filtering, aggregation,
  exploratory data analysis, and visual analytics

---

## Data Ethics

All analyses are performed on anonymized or aggregated data.
No personally identifiable information is included in this repository.
